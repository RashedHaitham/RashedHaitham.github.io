Serverless Performance Dashboard

Built a self-service leadership dashboard that replaced a manual monthly reporting process across all payment processors. The architecture is fully serverless , React frontend hitting API Gateway with JWT auth, backed by a Lambda that queries live monitoring APIs via MCP. Under the hood: a declarative collector engine where adding a new metric is a config entry, not a code change. The dashboard auto-computes year-over-year and period-over-period comparisons, tracks 30 goal targets with diff columns, handles ratio metrics like loan success rates through a formula engine, and ships with color-coded trend indicators, PDF export, and one-click email copy. What used to take hours of manual data collection now takes one click.
